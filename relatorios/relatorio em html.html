Prevendo Demanda de Estoque com Base em Vendas

Sobre

Projeto com feedback 02 do curso Big Data Analytics com R e Microsoft Azure Machine da Formao Cientista de Dados da Data Science Academy.

Dataset disponibilizado no kaggle pelo grupo Bimbo no desafio Grupo Bimbo Inventory Demand - Maximize sales and minimize returns of bakery goods.

Link do dataset: https://www.kaggle.com/c/grupo-bimbo-inventory-demand

Objetivo: Construir um modelo de machine learning para prever com preciso a demanda de estoque com base nos dados histricos de vendas.

Processo de Machine Learning

Separao de datasets de treino e teste

O dataset de trabalho o de treino, vamos baixar o de teste para realizao das previses:

teste <- fread('test.csv')

Treinamento do algoritmo

Modelo 1 - Regresso linear

Conforme visto, alm das variveis numricas, a nica categrica que parece ter alguma relao e/ou influncia no conjunto de dados a Canal_ID.

Por isso, alm das numricas, vamos utiliz-la na primeira verso do nosso modelo:

modelo1 <- lm(Demanda_uni_equil ~ Venta_uni_hoy + Venta_hoy + Dev_uni_proxima + Dev_proxima + Canal_ID, data = df_geral)

Previso

previsao1 <- predict(modelo1, data = teste)
